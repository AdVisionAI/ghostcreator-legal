Política de Privacidad

Última actualización: Mayo 2025

Resumen: GhostCreator no recopila, almacena ni transmite ningún dato personal a servidores externos. Todo funciona localmente en tu dispositivo.

1. Datos que manejas tú

Para funcionar, GhostCreator requiere que tú introduzcas las siguientes credenciales:

  • API Key de Google (Gemini AI)
  • Número de teléfono de WhatsApp
  • Token de acceso de Facebook Page
  • Credenciales OAuth de YouTube

Estos datos se guardan únicamente en tu dispositivo, en la carpeta %APPDATA%\GhostCreator\, y nunca abandonan tu equipo.

2. Datos que NO recopilamos

  • No existe ningún servidor del desarrollador que reciba datos.
  • No se usan analytics, telemetría ni rastreo de uso.
  • No se almacenan conversaciones de WhatsApp.
  • No se registra el contenido generado por la IA.

3. Comunicación con APIs de terceros

GhostCreator se comunica directamente desde tu dispositivo con:

  • Google Gemini API — para generación de texto e imágenes.
  • Meta / Facebook Graph API — para publicar contenido.
  • YouTube Data API v3 — para subir videos.
  • WhatsApp Web — para recibir y enviar comandos.

Estas comunicaciones están sujetas a las políticas de privacidad de cada plataforma, fuera del control del desarrollador de GhostCreator.

4. Seguridad

Tus credenciales se guardan en texto plano en %APPDATA%\GhostCreator\.env. Recomendamos no compartir esa carpeta ni hacer respaldos en servicios en la nube sin cifrar.

5. Menores de edad

GhostCreator no está dirigido a menores de 18 años.
